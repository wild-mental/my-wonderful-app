### 선택 근거
- Super-Calc은 영양제 가격 비교와 성분 검증이라는 **복잡한 과정**을 자동화
- 사용자(일반 소비자)는 비교 과정보다 **결과(최저가, 인정 여부)**에 관심
- "검색 한 번이면 끝"이라는 **단순화(Simplification)** 메시지가 핵심

## STEP 1. [공통] 핵심 코어 필수요소 점검

> 목표: 6개 모두 체크 ✅

### ① 히어로 섹션 (Hero Section) — 첫인상 3초

- [x] **명확한 헤드라인**: "영양제, 진짜 가격을 **3초 만에** 찾아드립니다"
  - ✅ 고객이 얻을 최종 이득(진짜 가격 발견)이 한 문장으로 요약됨
  - ✅ "3초"라는 구체적 수치로 즉시성 강조
- [x] **보조 설명 (Sub-head)**: "1일 단가 기준 최저가 비교부터 식약처 기능성 원료 인정 여부까지. 광고 없는 순수 데이터로, 현명한 선택을 도와드립니다."
  - ✅ 비교 방법(1일 단가)과 검증 방법(식약처 DB)이 구체적으로 명시
- [x] **직관적 이미지**: 검색바 형태의 인터랙티브 CTA + 성분명 키워드 자동 순환 애니메이션
  - ✅ 서비스의 핵심 행동(검색)을 직관적으로 시뮬레이션

### ② 행동 유도 (CTA) — 전환 장치

- [x] **동사형 버튼**: 
  - 상단 히어로: "검색" 버튼 + "서비스 바로가기"
  - 중단 갤러리: "지금 무료로 비교해보기"
  - 하단 최종: "무료로 영양제 비교하기"
  - ✅ 모두 동사형으로 행동을 유도
- [x] **반복 배치**: 
  - ✅ **상단** (히어로 검색바 + 헤더 링크)
  - ✅ **중단** (Result Gallery 하단 CTA 버튼)
  - ✅ **하단** (Final CTA 섹션)
  - 총 4회 이상 CTA 노출

### ③ 신뢰의 증거 (Social Proof) — 의심 제거

- [x] **숫자/로고 증명**:
  - 누적 비교 건수: 128,400+
  - 검증 완료 원료: 1,240
  - 월간 활성 사용자: 34,000+
  - 사용자 만족도: 4.8 / 5
  - ✅ 4가지 객관적 수치 지표로 신뢰 구축

**공통 필수요소 달성률: 6/6 (100%)**

## STEP 2. [선택] C 유형 전략 요소 점검

> 핵심 목표: "편의성"과 "결과물"

### 🅲 C 유형 선택 요소

- [x] **[Input-Output Diagram]**: "검색 한 번이면 끝" 섹션
  - 3단계 워크플로우 시각화: [성분명 검색] → [⚡ Super-Calc 엔진] → [✓ 결과 확인]
  - ✅ 복잡한 비교/검증 과정을 블랙박스화하고, 단순한 입력→출력 구조로 표현
  - 중앙 엔진 단계에 그래디언트 + 그림자로 시각적 강조

- [x] **[결과 갤러리 (Outcome Showcase)]**: "이런 결과를 받아보세요" 섹션
  - 실제 서비스의 비교 결과 카드를 고충실도(High-fidelity)로 재현
  - NMN 성분 3개 제품의 1일 단가 + 뱃지(최저가/인정/주의) 표시
  - 브라우저 크롬(창 프레임) 디자인으로 실제 사용 환경 시뮬레이션
  - ✅ "나도 이런 결과를 받을 수 있다"는 기대감 형성

- [x] **[Before & After (ROI)]**: "Super-Calc 이전 vs 이후" 섹션
  - 가격 비교 시간: ~~평균 47분~~ → **3초**
  - 성분 검증 정확도: ~~개인 판단~~ → **식약처 DB 기반**
  - 비교 기준: ~~총 가격 (불공정)~~ → **1일 단가 (공정)**
  - ✅ 다크 배경으로 시각적 대비 강화, 체크마크 아이콘으로 해결 완료 표시

**C 유형 전략 요소 달성률: 3/3 (100%)**

## STEP 3. [최종] 자가 피드백

### Q. 고객이 우리 랜딩페이지를 보고 가질 첫 번째 생각은?

> **"내가 먹는 영양제 가격이 적정한지 바로 확인할 수 있겠네."**

- ✅ 헤드라인이 문제(비싼 가격)와 해결(3초 비교)을 명확히 전달
- ✅ 검색바 형태의 CTA가 즉각적인 행동 유도
- ❌ "그래서 이게 뭔데?" → 해당 없음 (서비스 목적이 명확)
- ❌ "어디를 눌러야 해?" → 해당 없음 (검색바 + CTA 반복 배치)

### Q. 경쟁사 대비 우리 페이지만의 '한 방(Kick)'은?

- [ ] 압도적인 권위 (A)
- [ ] 놀라운 시연 영상 (B)
- [x] **확실한 효율성 수치 (C)** — "47분 → 3초" 시간 절감 대비

### Q. 기타 현재 페이지에서 수정하고 싶은 부분은?
- 히어로 문구가 반발짝 엇나감
- 검색창 클릭시 실제 서비스로 연결되는데 랜딩페이지에서 검색어까지 넘겨서 결과페이지에 표시해주는걸로 변경 필요
- 결과페이지 인터랙티브 섹션으로 고도화 필요
- "검색한번이면 끝" 섹션은 워크플로우 제시한 디자인이 의아해서 수정 개선 필요
